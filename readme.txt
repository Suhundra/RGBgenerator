Простий проект для регулювавння кожним кольором RGB стрічки окремо.
Основна мета проекту - розібратись з обробкою натиснення на тактові кнопки і створення власних методів для обробки звичайного натиснення та довгого.
Є кнопки переміщення по меню, збільшення\зменшення яскравості кожного потоку, рандомна генерація кольору, скидування всіх значень.

Працююча симуляція роботи на вокві - https://wokwi.com/projects/367299099925949441
